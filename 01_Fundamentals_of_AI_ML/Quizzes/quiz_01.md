### Question 1
Which of the following is accurate regarding the relationship between AI, ML, and Deep Learning?

A) AI is a subset of ML, which is a subset of Deep Learning.
B) Deep Learning is a subset of ML, which is a subset of AI.
C) ML and Deep Learning are completely separate fields from AI.
D) AI and ML are the same thing, but Deep Learning is different.

<details>
<summary><strong>View Answer</strong></summary>

**Correct Answer:** B

**Explanation:**
AI is the broad umbrella. Machine Learning is a subset of AI that learns from data. Deep Learning is a specialized subset of ML using neural networks.
</details>

### Question 2
A bank wants to detect fraudulent transactions in real-time as they happen. Which type of inference is most appropriate?

A) Batch Inference
B) Offline Inference
C) Real-time Inference
D) Scheduled Inference

<details>
<summary><strong>View Answer</strong></summary>

**Correct Answer:** C

**Explanation:**
Real-time inference is required for low-latency, immediate decision making like blocking a transaction while it is occurring.
</details>

### Question 3
You have a dataset of emails and you want to group them by topic without knowing the topics in advance. What type of ML is this?

A) Supervised Learning
B) Unsupervised Learning
C) Reinforcement Learning
D) Semi-supervised Learning

<details>
<summary><strong>View Answer</strong></summary>

### Question 5
What is 'Model Drift'?

A) When a model's accuracy degrades over time because real-world data changes.
B) When a model is moved from one AWS region to another.
C) When a model trains faster than expected.
D) When the cost of the model increases.

<details>
<summary><strong>View Answer</strong></summary>

**Correct Answer:** A

**Explanation:**
Model drift (or concept drift) happens when the statistical properties of the target variable change over time, making the model less accurate.
</details>
